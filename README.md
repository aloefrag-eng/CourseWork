# Социальная мини-платформа

Готовый проект для курсовой работы по теме:

**«Создание социальной мини-платформы (посты, лайки, комментарии — без backend)»**

## Что есть в проекте

- главная страница с лентой постов;
- отдельная форма добавления поста;
- страница о проекте;
- страница управления рекламой;
- лайки;
- комментарии;
- удаление постов;
- рекламный пост внутри ленты;
- правая рекламная карточка;
- хранение данных через `localStorage`.

## Запуск

1. Скачайте проект.
2. Распакуйте архив.
3. Откройте файл `index.html` в браузере.

Проект работает без сервера, без backend и без библиотек.

## Как пользоваться

### Главная страница
- просмотр ленты;
- лайк поста;
- открытие комментариев;
- добавление комментария;
- удаление поста;
- просмотр встроенного рекламного поста;
- просмотр правой рекламы.

### Добавление поста
1. Перейдите на страницу `form.html`.
2. Заполните поля:
   - автор;
   - текст поста;
   - ссылка на изображение.
3. Нажмите кнопку сохранения.
4. После этого откроется главная страница.

### Управление рекламой
1. Перейдите на страницу `ads.html`.
2. Измените параметры правой рекламы и рекламного поста.
3. Сохраните изменения.
4. Для возврата исходных значений используйте кнопку сброса.

## Хранение данных

При первом открытии автоматически создаются:
- тестовые посты;
- стандартная правая реклама;
- стандартный рекламный пост в ленте.

Все изменения сохраняются в `localStorage` браузера.
